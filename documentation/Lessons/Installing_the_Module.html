Installing the Module

The steps below describe how to install the extension/module.

Download the sugarcrm module package from SugarForge.org

Go to Admin->Module Loader

Select the zip file and click the upload button as shown above.

Once upload click the Install button for the uploaded package

If the installation was successfully the module should now display in the installed modules list

Installation is complete.
